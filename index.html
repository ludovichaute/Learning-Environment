Une liste a puces et une liste numeroté

Une liste a puces

  • Pour afficher une liste, commencez la ligne par une astérisque*, un moins - ou un plus +.
    Là encore, le choix n’a pas d’importance, mais il faut rester cohérent dans votre document.
  • Exemple:
    • Git
    • MarkDown
    • GitHub

Une liste numerotee

  • Pour afficher une liste ordonnée, commencez la ligne par un nombre suivit d’un point.
    Pour que votre markdown soit plus lisible, je vous conseille d’ordonner proprement votre liste.
  • Exemple:
    1. Git
    2. Markdown
    3. GitHub
